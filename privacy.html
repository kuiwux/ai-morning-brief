隐私政策

硅谷AI晨报

最后更新日期：2025年6月1日

📋 概述
硅谷AI晨报（以下简称"本应用"）尊重并保护您的隐私。 本隐私政策旨在向您说明我们如何收集、使用和保护您的信息。

一、信息收集

1.1 我们主动收集的信息

本应用不要求用户注册或登录，不主动收集任何个人身份信息（如姓名、邮箱、电话号码等）。

1.2 自动收集的信息

为提供基本的服务功能，本应用可能会自动收集以下非个人身份信息：

  • 设备信息：设备型号、操作系统版本，用于适配显示效果
  • 网络信息：IP 地址、网络类型，用于内容分发和性能优化
  • 使用数据：文章阅读偏好、访问时长，用于改善内容推荐服务
  • 缓存数据：为支持离线阅读，本应用使用 Service Worker 在您的设备上缓存文章内容

二、信息使用

我们收集的信息仅用于以下目的：

  • 提供和优化新闻资讯服务
  • 改善用户体验和界面设计
  • 支持离线阅读功能
  • 统计分析应用使用情况

三、推送通知

本应用支持浏览器推送通知功能。推送通知基于 Web Push API 实现：

  • 推送订阅信息（endpoint、公钥等）仅用于向您的设备发送通知
  • 您可随时在浏览器设置中关闭推送通知权限
  • 推送通知的 token 不会与任何第三方共享

四、数据存储

  • 本地存储：阅读偏好、缓存内容等数据存储在您的设备上
  • 服务器存储：推送订阅信息存储在我们的服务器上，用于发送通知
  • 所有数据存储均采用合理的加密和保护措施

五、第三方服务

本应用目前不使用以下第三方服务：

  • 第三方分析工具（如 Google Analytics）
  • 广告网络或广告投放平台
  • 社交媒体 SDK

如未来引入第三方服务，我们将更新本政策并通知用户。

六、Cookie 和类似技术

  • 本应用可能使用浏览器本地存储（localStorage）来保存您的阅读偏好
  • 不使用第三方 Cookie 或追踪 Cookie
  • Service Worker 缓存不会收集个人身份信息

七、儿童隐私

本应用不面向 13 岁以下儿童。我们不会故意收集儿童的个人信息。如果我们发现不慎收集了儿童的信息，将立即删除。

八、信息安全

我们采取合理的技术和管理措施保护您的信息安全，包括但不限于：

  • 数据传输加密（HTTPS）
  • 服务器访问控制
  • 定期安全审查

九、您的权利

您享有以下权利：

  • 随时清除浏览器缓存和本地存储数据
  • 在浏览器设置中关闭推送通知
  • 卸载应用以删除所有本地数据
  • 如对我们的数据处理有疑问，可通过下方联系方式咨询

十、政策更新

我们可能会不时更新本隐私政策。更新后的政策将在本页面发布，重大变更将通过应用内通知或推送通知告知用户。

十一、联系方式
